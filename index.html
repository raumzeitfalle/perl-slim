Perl Slim.

This is an implementation of the Fitnesse Slim protocol (http://www.fitnesse.org/FitNesse.UserGuide.WritingAcceptanceTests.SliM) for Perl. Combined with fitnesse this allows tests authored on the fitnesse wiki to run through Perl test fixtures against Perl application code.

Installing

  • Download and install fitnesse (http://www.fitnesse.org/FitNesseDownload).
  • Download perl-slim and unpack/unzip to location you desire. Since you may install different instances of fitnesse and/or versions of fitnesse over time, you may want to keep perl-slim directory independent of your fitnesse directories.
  • Unzip the PerlTutorialsAndExamples.zip file included in perl-slim, and copy the resulting directory to the FitNesseRoot folder of your target fitnesse instance.

  • Start fitnesse, and edit the home page of fitnesse to add a link to the PerlTutorialsAndExamples folder, adding this text to page should work to create a link:

    !3 PerlTutorialAndExamples

  • Navigate into the PerlTutorialAndExamplesPage, and read the SettingUpPerlFitnesse child page for instructions on making the Perl sample fixtures runnable.

Help Using Perl Slim.

The PerlTutorialAndExamplesPage contains a number of runnable examples and snippets of fixture code. The full fixture code can be found in perl-slim under lib/Slim/SampleFixtures.
