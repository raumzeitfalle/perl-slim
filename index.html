Perl Slim.

This is an implementation of the Fitnesse Slim protocol (http://www.fitnesse.org/FitNesse.UserGuide.WritingAcceptanceTests.SliM) for Perl. Combined with fitnesse this allows tests authored on the fitnesse wiki to run through Perl test fixtures against Perl application code.
